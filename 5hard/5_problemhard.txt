문제 : 수연이의 빵이 그립다...죽빵말고
문제명 :
찬희에겐 여자친구 갓자친구들이 승빈이에겐 여자친구 트둥이들이 있다. 뚜레쥬르 아르바이트생이였던 수연이는 사장으로 부터 가계를 받아 사장이 되었다. 부자가 된 수연은 동아리방에 있는 찬희와 승빈이에게 종류가 같은 단팥빵을 1000개 주었다. 이 둘은 빵때문에 싸우게되었다. 반절씩 나눠가지면 되지만 찬희에겐 6명의 여자친구가 있고 승빈이에겐 9명의 여자친구들이 있다. 수연이는 싸우는걸 말리고 게임을 제안했다. 수연이가 N을 말해 1~N까지 게임을 진행한다. 1000000000000개의 빵을 찬희와 승빈이는 번갈아가면서 가져간다, 이 때 빵은 1개, 3개, 4개만 가져 갈 수 있다. 마지막 빵을 가져가는 사람이 게임을 이기게 되고 빵 1000000000000개를 획득하게 된다. 찬희와 승빈이가 완벽하게 게임을 했을 때, 게임의 승자를 구하는 프로그램을 만드시오. 게임은 찬희가 먼저 시작한다.

입력설명 
첫째줄에 테스트케이스 T가 주어진다. T는 1이상 100이하이다.
그 후 T줄에 걸쳐 N이 주어진다 N은 1이상 1,000,000,000,000이하이다. 

출력
찬희가이기면 Buddy 승빈이가 이기면 Once

입력샘플
7
7
5
3
1
2
4
6

출력샘플
Once
Buddy
Buddy
Buddy
Once
Buddy
Buddy

코드
#include <iostream>
using namespace std;

long long N;
int main() {
	cin >> N;
	if (N % 7 == 0 || N % 7 == 2) cout << "Once";
	else cout << "Buddy";
	return;
}
